# Taller de TypeScript

Este proyecto es un taller de TypeScript que incluye varios programas de ejemplo para trabajar con entradas de usuario, reservas y ventas en consola. Contiene scripts de entrenamiento escritos en TypeScript para practicar lógica de negocio y manejo de datos en Node.js.

## Instalación

Instala las dependencias con npm:

```bash
npm install
```

## Ejecución

Ejecuta el archivo principal o cualquier script TypeScript usando Bun o Node según tu preferencia.

Con Bun:

```bash
bun run index.ts
```

Con Node y TypeScript instalado globalmente:

```bash
npx ts-node index.ts
```

## Archivos principales

- `index.ts` — punto de entrada del proyecto
- `Boleteria.ts` — sistema de venta de boletas de cine
- `Sistema_de_Reserva.ts` — sistema de reservas de hotel
- `Fila_de_Banco.ts` — ejemplo de gestión de filas en consola

## Requisitos

- Node.js o Bun
- TypeScript (se instala como dependencia)

## Notas

El proyecto está configurado para TypeScript con `tsconfig.json` y utiliza definiciones de tipo de Node para ejecutar código de consola correctamente.
